# YouTube Slide Extractor 

A simple Python desktop application to extract key slides from YouTube videos. Useful for summarizing lecture content, presentations, and tutorials.

##  Features

* GUI built with Tkinter for ease of use
* Downloads and processes YouTube videos
* Detects and extracts unique slide frames
* Saves extracted slides as images
